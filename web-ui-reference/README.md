# Web UI Reference (PlayO Style)

This is the **design reference** for Clipora's UI, built as a React web app.

## Purpose
- Visual design reference for the React Native mobile app
- Can be run for quick UI previews
- NOT the production app (use `/mobile` for that)

## Run Locally
```bash
npm install
npm run dev
```

## Note
The main mobile app is in `/mobile` (React Native + Expo).
This web version is for design reference only.
